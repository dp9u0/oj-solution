# 41. First Missing Positive

## Description

Given an unsorted integer array, find the smallest missing positive integer.

Note: Your algorithm should run in O(n) time and uses constant extra space.

## Example

```javascript
Input: [1,2,0]
Output: 3
```

```javascript
Input: [3,4,-1,1]
Output: 2
```

```javascript
Input: [7,8,9,11,12]
Output: 1
```
